Bilingual Chatbot

By Technical Team
January 1, 2023

This is a linguistic based chat bot built using stories and rules in rasa environment. It basically use NLU to understand the meaning of the user's input. Primarily focused on machine reading comprehension, NLU gets the chatbot to comprehend what a body of text means. The interesting feature is we can chat using 'telugu' language as it is a rule based bot.
